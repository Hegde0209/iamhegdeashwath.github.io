Megatron 3000 Database Management System

This is a lighter version of commercially available Relational Database Management Systems in today's market which was implemented as three modules. The entire project was implemented in C Programming Language.

  • Storage Manager - This is capable of reading blocks from a file on disk into memory and writing blocks from memory to a file on disk.
  • Buffer Manager – This manages a fixed number of pages in memory that represent pages from a page file managed by the storage manager using the replacement strategies like FIFO, LRU and LFU to fetch blocks from the Hard Disc in case the Buffer is full.
  • Record Manager - This handles tables with a fixed schema. Clients can insert records, delete records, update records, and scan through the records in a table. This module make use of buffer manager and storage manager.
